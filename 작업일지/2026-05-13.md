## 1. 인수테스트 시나리오 S-07 수정 — 수동 스캔 → 즉시 검사

### 기존 상태
`문서/인수테스트_시나리오.md` S-07 제목이 "수동 스캔 실행"이었고, 내부에서 존재하지 않는 "수동 스캔" 버튼을 클릭하도록 안내하고 있었음. 사후 조건도 "로그에 기록된다"로 잘못 작성됨. 예외 흐름도 실제 동작과 다른 내용이었음.

### 문제점
실제 UI에는 "수동 스캔" 버튼이 없음. "즉시 검사" 버튼이 정확한 명칭이며, 이 버튼은 단순 스캔이 아니라 baseline 등록 완료 후 자동으로 감시까지 시작하는 기능임. 또한 즉시 검사는 AI 분석을 하지 않으므로 "이상 파일 발견 시 알림 생성"은 사실과 다름.

### 변경 내용
- S-07 제목: "수동 스캔 실행" → "즉시 검사 실행"
- 목적: baseline 등록 후 자동 감시 시작으로 정확하게 수정
- 사후 조건: "스캔 완료 후 로그 기록" → "스캔 완료 후 자동으로 실시간 감시 시작"
- 시나리오 3단계 버튼명: "수동 스캔" → "즉시 검사"
- 10단계: "이상 파일 발견 시 알림 생성" 제거 → 실제 완료 모달 동작(파일 수 표시, 감시 자동 시작)으로 교체
- 예외 흐름: 감시 폴더 미등록 시 경고 메시지로 교체
- 목록 테이블 및 체크리스트의 "S-07 수동 스캔 실행" → "S-07 즉시 검사 실행"

## 2. 시연 대본 작성

### 기존 상태
시연 영상을 위한 대본 문서 없음.

### 문제점
실제 사용자 시연 영상 촬영 시 어떤 순서로, 어떤 대사와 행동을 해야 하는지 정리된 문서 필요.

### 변경 내용
`문서/시연_대본.md` 신규 작성.
- **PART 1 (관리자, 장면 1~10)**: 로그인 → 시스템 상태 모니터링 → 세션 관리 → 로그 관리 → 알림·탐지 관리 → 공지사항 등록/삭제 → 피드백 관리 → 사용 가이드 편집 → 로그아웃
- **PART 2 (일반 사용자, 장면 11~31)**: 앱 실행 → 감시 폴더 등록 → 즉시 검사 → 알림 목록/상세/통계 → 이벤트 로그/TOP10/확장자 분포 → 공지사항 → 설정 전체(감시 폴더/예외/알림방식/이메일/스케줄/업데이트/피드백/가이드) → 감시 중지
- 각 장면마다 `[행동]` (조작 지시)과 `(나레이션)` (대사) 분리 표기
- 마지막에 관리자/사용자 파트별 체크리스트 포함
