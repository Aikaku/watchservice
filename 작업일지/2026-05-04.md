## 3. 컴퓨터 시작 시 자동 실행 설정 기능 추가

### 기존 상태
```javascript
// main.js — app.whenReady()
if (app.isPackaged && !app.getLoginItemSettings().openAtLogin) {
  app.setLoginItemSettings({ openAtLogin: true }); // 첫 실행 시 무조건 자동 등록
}
```
트레이 메뉴에만 토글이 있고 설정 페이지에는 노출되지 않았음.

### 문제점
- 앱 설치 즉시 시작 프로그램 자동 등록됨 (사용자 동의 없음)
- 끄려면 트레이 메뉴를 찾아야 해서 직관적이지 않음

### 변경 내용
**`main.js`** — 강제 자동 등록 코드 삭제, IPC 핸들러 추가
```javascript
ipcMain.handle('get-login-item', () => app.getLoginItemSettings().openAtLogin);
ipcMain.handle('set-login-item', (_event, enabled) => {
  app.setLoginItemSettings({ openAtLogin: !!enabled });
  tray.setContextMenu(buildContextMenu());
  return app.getLoginItemSettings().openAtLogin;
});
```

**`preload.js`** — renderer에 API 노출
```javascript
getLoginItem: () => ipcRenderer.invoke('get-login-item'),
setLoginItem: (enabled) => ipcRenderer.invoke('set-login-item', enabled),
```

**`SettingNotifyPage.jsx`** — "컴퓨터 시작 시 자동 실행" 토글 추가
- 페이지 진입 시 현재 값 로드, 토글 변경 즉시 시스템 적용
- Electron 환경에서만 표시 (`isElectron` 조건부 렌더링)

## 4. 앱 아이콘 파일 생성

### 기존 상태
`watchservice_electron/assets/`에 아이콘 파일 없음.

### 문제점
빌드는 되지만 앱 아이콘 미표시 (작업표시줄, Dock, Finder, 트레이 모두).

### 변경 내용
프로젝트 루트의 `아이콘.png`를 기반으로 플랫폼별 아이콘 변환.

- `icon.png` (1.1MB) — 트레이 아이콘용 원본 복사
- `icon.ico` (59KB) — Python PIL로 16/32/48/256px 멀티사이즈 ICO 생성 (Windows)
- `icon.icns` (1.6MB) — `sips` + `iconutil`로 16~512px @1x/@2x ICNS 생성 (macOS)
