# 2026-03-09 작업 일지

## 1. P0: 관리자 비밀번호 BCrypt 해싱

평문 비밀번호 비교 방식을 BCrypt 검증으로 전환.

| 파일 | 변경 내용 |
|------|----------|
| `build.gradle` | `spring-security-crypto` 의존성 추가 |
| `admin/AdminAuthService.java` | `BCryptPasswordEncoder` 도입. `ADMIN_PASSWORD`가 `$2a$`/`$2b$`로 시작하면 BCrypt 검증, 아니면 평문 비교 (기존 배포 호환 유지) |
| `src/main/resources/application.yml` | BCrypt 해시 설정 방법 주석 추가 |
| `start.sh` | `JAVA_HOME=Java 17` 명시 — Gradle 8.x가 Java 25 미지원, 시스템 기본 Java가 25이므로 명시 필요 |

### BCrypt 해시 교체 방법
```java
// Java 코드로 해시 생성
new BCryptPasswordEncoder().encode("yourPassword")
```
생성된 해시를 `.env`의 `ADMIN_PASSWORD` 값으로 교체하면 즉시 적용됨.

## 빌드 결과

```
JAVA_HOME=Java 17로 ./gradlew build -x test --no-daemon → BUILD SUCCESSFUL
```
