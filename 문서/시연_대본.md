### 장면 5 — 랜섬웨어 탐지 시연 *(방법 A / B 중 하나 선택)*

> **방법 A — 직접 파일 조작 (WARNING 수준 유도)**

`[행동]` 앱 창을 화면 한쪽에 두고, Finder(macOS) 또는 탐색기(Windows)에서 감시 폴더를 열어 나란히 배치한다

> (나레이션) "감시가 활성화된 상태에서 폴더 안 파일을 직접 변경해보겠습니다."

`[행동]` 감시 폴더 안에서 텍스트 파일을 5~10개 빠르게 새로 만들고, 기존 파일 2~3개를 열어 내용을 수정한 뒤 저장한다

> (나레이션) "짧은 시간 안에 여러 파일이 변경되면 시스템이 이상 징후로 판단합니다."

`[행동]` 앱 화면에 경고 팝업 또는 상태 표시가 바뀌는 것을 보여준다

> (나레이션) "탐지됐습니다. 대시보드 상태가 바뀌고 알림이 발생한 것을 확인할 수 있습니다."

---

> **방법 B — 테스트 스크립트 실행 (DANGER 수준 확실 유도)**

`[행동]` 터미널을 열고 아래 명령어를 실행한다 (감시 폴더 경로를 실제 경로로 교체)

```bash
python3 -c "
import os, random
folder = '/감시폴더/경로'
for i in range(20):
    path = os.path.join(folder, f'test_{i}.enc')
    with open(path, 'wb') as f:
        f.write(bytes([random.randint(0,255) for _ in range(4096)]))
print('done')
"
```

> (나레이션) "테스트 스크립트로 고엔트로피 파일 20개를 빠르게 생성합니다. 랜섬웨어가 파일을 암호화할 때와 유사한 패턴입니다."

`[행동]` 스크립트 실행 후 앱 화면을 바라본다

> (나레이션) "위험 등급 탐지가 발생했습니다. 대시보드 상태가 '위험'으로 바뀌고 AI 대응 가이드가 생성됩니다."

`[행동]` 탐지 후 생성된 테스트 파일들을 삭제한다

```bash
rm /감시폴더/경로/test_*.enc
```
