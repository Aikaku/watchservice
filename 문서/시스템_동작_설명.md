# WatchService Agent — 시스템 동작 설명

> 비전공자도 이해할 수 있도록 작성한 동작 원리 설명 문서입니다.

---

## 한 줄 요약

> **"컴퓨터 안의 파일들이 이상하게 바뀌면 즉시 알려주는 보안 프로그램"**

---

## 랜섬웨어가 뭔가요?

랜섬웨어는 컴퓨터에 몰래 들어와 **파일을 암호화(잠금)해버리고 돈을 요구**하는 악성 프로그램입니다.

```
정상 파일 → 랜섬웨어 감염 → 열 수 없는 파일로 변환
                              "돈 내면 풀어줌" 메시지 등장
```

피해를 막으려면 **암호화가 시작되는 순간 최대한 빨리 탐지**하는 것이 핵심입니다.

---

## WatchService는 어떻게 탐지하나요?

### 핵심 아이디어: "행동을 본다"

바이러스 백신은 "이 파일이 악성 코드인지" 파일 자체를 검사합니다.  
WatchService는 다릅니다. 파일 내용 대신 **"파일이 어떻게 바뀌고 있는가"** 를 봅니다.

> 랜섬웨어는 아무리 새로운 변종이어도 결국 파일을 **대량으로 빠르게 암호화**합니다.  
> 이 "행동 패턴"은 숨길 수 없습니다.

---

## 단계별 동작 흐름

### 1단계 — 감시 시작

사용자가 "감시 폴더"를 지정하면, WatchService가 해당 폴더를 **24시간 실시간 모니터링**합니다.

```
[감시 폴더 지정]
  내 문서 폴더
  바탕화면
  중요 업무 폴더  ← 여기서 파일이 생성/수정/삭제될 때마다 감지
```

---

### 2단계 — 파일 이벤트 수집 (3초 단위)

파일 변화가 감지되면 3초 동안의 이벤트를 모아서 9가지 수치를 계산합니다.

| 수치 | 의미 | 랜섬웨어 시 |
|------|------|------------|
| 파일 읽기 횟수 | 얼마나 많이 열었나 | 매우 많음 |
| 파일 쓰기 횟수 | 얼마나 많이 덮어썼나 | 매우 많음 |
| 파일 삭제 횟수 | 얼마나 지웠나 | 많음 |
| 파일 이름 변경 횟수 | 확장자가 바뀌었나 | 많음 |
| 암호화 의심 파일 수 | 갑자기 복잡해진 파일 수 | 많음 |
| 변경된 파일 수 | 몇 개나 바뀌었나 | 매우 많음 |
| 의심 확장자 감지 | .locked 같은 이상한 확장자 | 있음 |
| 엔트로피 변화 | 파일 내용이 얼마나 "복잡"해졌나 | 급격히 높아짐 |
| 파일 크기 변화 | 크기가 얼마나 달라졌나 | 달라짐 |

> **엔트로피**란?  
> 파일 내용의 "무작위성" 정도입니다.  
> 일반 텍스트는 엔트로피가 낮고, 암호화된 파일은 엔트로피가 매우 높습니다.  
> 랜섬웨어가 파일을 암호화하면 엔트로피가 갑자기 확 올라가므로 핵심 신호로 활용합니다.

---

### 3단계 — AI 판정

수집한 9개 수치를 AI 모델(XGBoost)에 전달합니다.

```
9개 수치 입력
     ↓
  XGBoost AI
     ↓
랜섬웨어 확률 계산
```

AI는 3만 건의 데이터로 학습해 정상/랜섬웨어 행동 패턴의 차이를 학습했습니다.

**판정 기준:**

```
랜섬웨어 확률 70% 이상 → 🔴 DANGER  (즉시 경보)
랜섬웨어 확률 50~70%  → 🟡 WARNING (주의)
랜섬웨어 확률 50% 미만 → 🟢 SAFE    (정상)
```

---

### 4단계 — DANGER 탐지 시 대응

DANGER 판정이 나오면 동시에 3가지 일이 일어납니다.

```
DANGER 탐지
    ├── 1. 전체화면 긴급 경보창 표시
    ├── 2. 이메일 알림 발송 (설정한 주소로)
    └── 3. Gemini AI가 대응 방법 생성
             예) "즉시 네트워크를 차단하세요"
                 "외장 드라이브를 연결하지 마세요"
```

---

### 5단계 — 대시보드 확인

모든 탐지 기록은 대시보드에서 확인할 수 있습니다.

- 언제 / 어떤 파일이 / 얼마나 위험했는지 이력 조회
- DANGER / WARNING / SAFE 통계 그래프
- 오탐(정상인데 경보) 신고 기능

---

## 전체 흐름 한눈에 보기

```
파일 변화 감지
     ↓
3초 동안 이벤트 수집
     ↓
9개 행동 수치 계산
     ↓
XGBoost AI 판정
     ↓
SAFE ──────────────────────── 기록만 저장
WARNING ───────────────────── 경고 알림 저장
DANGER ────┬─────────────────── 긴급 경보창
           ├─────────────────── 이메일 발송
           └─────────────────── Gemini AI 대응 가이드
```

## 자주 묻는 질문

**Q. 인터넷이 없어도 되나요?**  
A. 기본 탐지(XGBoost)는 인터넷 없이 동작합니다. Gemini AI 대응 가이드는 인터넷이 필요합니다.

**Q. 컴퓨터가 느려지지 않나요?**  
A. 파일 전체를 검사하지 않고 행동 수치만 계산하므로 시스템 부하가 매우 낮습니다.

**Q. 오탐(잘못된 경보)은 없나요?**  
A. 대용량 파일 복사나 압축 작업 시 WARNING이 뜰 수 있습니다. 대시보드에서 "오탐 신고" 버튼으로 기록할 수 있습니다.

**Q. 랜섬웨어를 막아주나요?**  
A. 차단 기능은 없습니다. **탐지 후 사용자에게 즉시 알려** 피해 확산 전에 대응하도록 돕는 것이 목적입니다.
