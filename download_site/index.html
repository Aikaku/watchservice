설치 방법

1
인스톨러 다운로드 운영체제에 맞는 버튼을 클릭해 설치 파일을 다운로드합니다.
2
설치 실행 Windows: .msi 파일 실행 → 안내에 따라 설치
macOS: .dmg 파일 열기 → Applications 폴더로 드래그
3
앱 실행 시작 메뉴(Windows) 또는 응용 프로그램(macOS)에서 WatchService Agent를 실행합니다.
4
감시 폴더 설정 대시보드 → 설정 → 감시 폴더에서 보호할 폴더를 추가하면 실시간 탐지가 시작됩니다.
⚠ 주의
macOS에서 "확인되지 않은 개발자" 경고가 표시될 수 있습니다.
시스템 설정 → 개인 정보 보호 및 보안 → "그래도 열기"를 클릭하면 정상 실행됩니다.
