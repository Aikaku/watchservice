# WatchService Agent 설치 가이드

---

## Windows 설치

### 1. 인스톨러 다운로드
`.msi` 파일을 다운로드합니다.

### 2. 설치 실행
- `.msi` 파일을 더블클릭합니다.
- "이 앱이 변경할 수 있도록 허용하시겠습니까?" → **예**
- 설치 마법사 안내에 따라 진행합니다.

> **SmartScreen 경고 발생 시:**
> "추가 정보" → "실행" 클릭

### 3. 앱 실행
- 바탕화면 또는 시작 메뉴에서 **WatchService Agent** 실행
- 오른쪽 하단 시스템 트레이에 아이콘이 나타나면 정상 실행

### 4. 제거
제어판 → 프로그램 추가/제거 → WatchService Agent → 제거

---

## macOS 설치

### 1. 인스톨러 다운로드
`.dmg` 파일을 다운로드합니다.

### 2. 설치
- `.dmg` 파일을 더블클릭하여 마운트
- 열린 창에서 **WatchService Agent** 아이콘을 **Applications** 폴더로 드래그

### 3. 최초 실행 (보안 경고 처리)
macOS는 App Store 외 앱을 처음 실행할 때 경고를 표시합니다.

**방법 A (우클릭):**
1. Finder에서 WatchService Agent 아이콘 **우클릭** → **열기**
2. "확인되지 않은 개발자" 경고에서 **열기** 클릭

**방법 B (시스템 설정):**
1. 앱 실행 시 차단됨 → 시스템 설정 → 개인 정보 보호 및 보안
2. "WatchService Agent 열기가 차단됨" 옆 **그래도 열기** 클릭

### 4. 앱 실행
- 응용 프로그램 폴더 또는 Launchpad에서 **WatchService Agent** 실행
- 상단 메뉴바에 아이콘이 나타나면 정상 실행

### 5. 제거
Applications 폴더에서 WatchService Agent를 휴지통으로 이동

---

## 초기 설정

앱 실행 후 대시보드가 열립니다.

### 감시 폴더 추가
1. 대시보드 → **설정** → **감시 폴더**
2. **+ 폴더 추가** 클릭 → 보호할 폴더 선택
3. 실시간 탐지 시작

### 관리자 페이지
- 초기 계정: `admin` / `123456789`
- 로그인 후 비밀번호 변경 권장

## 문제 해결

| 증상 | 해결 방법 |
|------|----------|
| 앱이 시작되지 않음 | 트레이 아이콘 확인. 없으면 작업 관리자에서 프로세스 종료 후 재실행 |
| 대시보드 흰 화면 | 서비스 시작 대기 중 (최대 60초). 그래도 안 되면 앱 재시작 |
| 탐지가 작동하지 않음 | 설정 → 감시 폴더가 올바르게 설정되어 있는지 확인 |
| AI 분석 오류 | 인터넷 연결 확인 (AI 서버는 클라우드에서 실행됨) |
| macOS 권한 오류 | 시스템 설정 → 개인 정보 보호 → 파일 및 폴더 → WatchService Agent 허용 |

|------|----------|
| OS | Windows 10 64bit / macOS 11 이상 |
| RAM | 512 MB 이상 여유 |
